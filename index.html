Personal projects

Mutation Station

Many computational mutagenesis tools allow batch submission of mutations. Only problem is it can be a nightmare to produce the list of mutations in the required format without some basic scripting skills. This tool takes the pain out of this situation by generating mutation inputs for a number of popular services and tools, from either a protein sequence or a PDB file. All Javascript/HTML/CSS - all client side.

ProtViewer

Warning: very much a work in progress!

This tool allows one to examine various functional/sequence annotations from UniProt in a structural context including mutations, active sites and more. The annotations are then displayed on the structure, and the interactions and neighbourhoods of interesting residues/sites can be visualised interactively. The NGL viewer library is used for visualisation of protein structure. Pure Javascript/HTML/CSS - all runs client side. There are a number of bugs/issues but they are being actively worked on when I have the time.
