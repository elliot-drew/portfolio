About me

I'm Elliot Drew, a postdoctoral researcher at Queen Mary, University of London. I do bioinformatics research - developing new tools and algorithms for the analysis of biological data - and work on personal projects in my spare time.

I got my undergrad degree in Biochemistry from University of Bristol, an MSc in Bioinformatics and Theoretical Systems Biology from Imperial College London, and finally my PhD in Structural Biology from Birkbeck College, University of London.

I've been a postdoctoral researcher at Queen Mary University, University of London since 2017. I develop and maintain a number of websites and tools that serve the structural biology community, with a particular focus on Circular Dichroism Spectroscopy. Occasionally I will detach myself from my computer and go into a wet-lab to collect data.

Personal and work projects

Below you can see a few of my personal projects, and some that I have produced during my PostDoc at QMUL. Code for personal projects can be found at my github profile.

StockSim

Work in progress - One page React application. Generates fake stock data using a Monte Carlo simulation model. Buy and sell shares as prices are generated and try and make a profit. Will be adding in more complex features to the model e.g. events.

ProtViewer

Warning: very much a work in progress!

This tool allows one to examine various functional/sequence annotations from UniProt in a structural context including mutations, active sites and more. The annotations are then displayed on the structure, and the interactions and neighbourhoods of interesting residues/sites can be visualised interactively.

monte-carlo-stock-api

Python REST API using Flask and Swagger. One endpoint - generates randomly generated fake stock data for a number of steps in OHLC candlestick format given user supplied parameters. Repository can be found here.

alphafold-disorder-predictor

Command line Python implementation of the disorder prediction approach outlined by Norman Davey and Balint Meszaros in this github repo. Both approaches (pLDDT and RSA based) are implemented. You just need an Alphafold PDB model file and/or a DSSP file derived from said model.

WikiPlinker

This stupidly named JavaScript library makes it easy to connect any element on a webpage to pages in Wikipedia, using simple to use and understand custom data- attributes. Currently this can be done by explicitly defining the title of the page or by searching for a term and returning the top hits, which are displayed as a toggleable tooltip on the web page.

react-disease-map

Repository for a React project I made to display disability-adjusted life years statistics for countries around the world.

SimPy Colab Notebooks

This is a repo where I will be putting colab notebooks produce while I test out/learn the SimPy Python library for work. SimPy is a process-based simulation framework which makes it super easy to simulate complex systems with discrete events. I will be adding to this over time.

Mutation Station

Many computational mutagenesis tools allow batch submission of mutations. Only problem is it can be a nightmare to produce the list of mutations in the required format without some scripting skills. This tool takes the pain out of this situation by generating mutation inputs for a number of popular services and tools, from either a protein sequence or a PDB file. All Javascript/HTML/CSS - all client side.

PDBMD2CD (Postdoc)

PDBMD2CD allows for the prediction of circular dichroism spectra from the atomic coordinates of proteins. The algorithm is significantly faster and more accurate than the previous iteration (PDB2CD), allowing for batch submission of structures. A number of analysis tools are provided, including spectra clustering and ensemble generation through fitting to experimental spectra if provided.

2StrucCompare (Postdoc)

2StrucCompare is a webserver whose primary aim is to visualize subtle but functionally important differences between two related protein structures, either of the same protein or related homologues, with similar or functionally different tertiary structures.
